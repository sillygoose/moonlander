After 40 years the classic computer game Moonlander has returned and it could not have been possible without the help of a number of individuals and open source software projects.  These are the people and projects that made the iOS version of Moonlander possible:

 

Jack Burness
GT40

Jack is the original author of Moonlander and single handedly created one of the earliest and most entertaining computer video games of all time.

 

 

Rick Naro
iOS

 

I had the pleasure to play Moonlander back in the mid-1970’s and when I decided to tackle iOS programming I thought it would be great to re-introduce Moonlander to a new generation of gamers that have never seen the classic simulation.  Hopefully my new iOS programming skills are more useful than my new found PDP‑11 and MACRO‑11 skills.

 

 

Bob Supnik
simh (PDP‑11)

This project would not have been possible without the work of the simh project.  This open source software made it possible to run all the original software, confirming the iOS version was a faithful copy of the original Moonlander.  A list of simh contributors can be found at:

http://simh.trailing-edge.com

 

 

Phil Bunde
simh (VT11)

Phil’s contribution of a X11 simulation of the VT11 hardware allowed Moonlander to run on a Mac. I have some VT11 bugs fixes should you ever want them.

http://www.ultimate.com/phil/xy/

 

 

Eric

Eric has a web page devoted to the GT40 and Moonlander and was an invaluable source for the binaries and source code to run Moonlander on modern hardware.

http://www.brouhaha.com/~eric/retrocomputing/dec/gt40/
