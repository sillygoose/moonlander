You are playing Lunar Lander on a Teletype that contains a printer and keyboard unit.  Because of size constraints the simulated keyboard is limited to the keys accepted by Lunar Lander game:

Numeric keys to enter the burn rate, which may be 0 or in the range 8 to 200.

This key is a shortcut to enter the string ÔYESÕ when asked if you wish to play another game.

This key is a shortcut to enter the string ÔNOÕ when you are finished playing Lunar Lander.

If you change your mind you can use the RUBOUT key to delete one keystroke from the input buffer each time it is pressed.

All user input is followed by the RETURN key to continue processing by the Lunar Lander program.

 

 

Above the printer output is a debugging console with the following controls:

Immediately ends the game and returns you to the Lunar Lander menu.

This button is used to enable and disable Focal statement debugger.  When statement tracing is enabled, the Focal source code appears in a slide-down window allowing you to observe the code as it executes.

Use this slider to control how fast the Focal statements are executed.

Teletypes are noisy, use this slider to control the volume of the Teletype output.
